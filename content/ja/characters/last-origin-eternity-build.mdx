<Section icon={Target} title="最適なステータス配分と上限">

成功する**ラストオリジン エタニティのビルド**の基盤は、ステータス配分です。ラストオリジン R+では、ヒットとクリティカルの計算が正確であるため、ポイントを無駄にすると、終盤のストーリーステージやイベントでのパフォーマンスが著しく低下する可能性があります。2026年、重攻撃手のメタは「最大上限」よりも「信頼性」を重視しています。ミスしたりクリティカルが出なかったりすれば、ダメージを与えることはできません。

以下の表は、レベル100で目指すべき目標ステータスを示しています。これらの値には、装備とパッシブスキルによるバフが含まれていることに注意してください。

| ステータスタイプ | 目標値 | 優先度 | 理由 |
| :--- | :--- | :--- | :--- |
| **クリティカルヒット** | **100%** | **最高** | すべての攻撃が最大ダメージ倍率を発動するようにする |
| **命中** | **250% - 300%** | **高** | チャプター8+の高回避敵に当てるために必要 |
| **攻撃力 (ATK)** | **残りのポイント** | **中** | 上限に達したらAOEダメージをスケールさせる |
| **HP / 防御** | **任意** | **低** | 特定のソロファーミングビルドでのみ投資 |
| **速度 / AP** | **装備経由** | **N/A** | ステータス/ポイントでスケールさせるのは難しいため、装備を使用 |

<Callout type="warning" title="ステータス計算に関する注意">
ポイントを費やす前に、必ず彼女のパッシブスキルを確認してください。エタニティは、部隊にいるだけでクリティカルヒット率が10〜20%増加することがよくあります。過剰に投資すると、それらのポイントは事実上無駄になります。すべてのアクティブバフを含めて「合計」100%を目指しましょう。
</Callout>

</Section>

<Section icon={Zap} title="エタニティに最適な装備とモジュール">

ラストオリジン R+の装備は、チップ（2）、OS（1）、特殊モジュール（1）の4つのスロットに分かれています。エタニティの場合、彼女の低い速度を補いつつ、彼女の純粋なパワーを倍増させる装備が必要です。現在の2026年5月バージョンでは、いくつかの新しいR+限定アイテムが「ベストインスロット」の推奨事項を変更しています。

  <Step num="1" title="チップスロット: 命中とパワー">

  </Step>
  <Step num="2" title="OSスロット: 対タイプ最適化">
    「対軽装」または「対重装」OSモジュールが一般的に最適です。エタニティはAOEスペシャリストであるため、現在クリアしているステージの敵の大部分と一致するOSを使用してください。
  </Step>
  <Step num="3" title="特殊モジュール: エナジーパックまたはスコープ">
    **エナジーパック**は、**ラストオリジン エタニティのビルド**の標準です。毎ラウンド開始時に追加のAPを提供し、彼女の基本速度よりもはるかに早く行動できるようにします。

<Section icon={Swords} title="9マス部隊編成とシナジー">

ラストオリジン R+は9マスの戦場によって定義されます。**ラストオリジン エタニティのビルド**を正しく配置することが、完全な掃討と部隊壊滅の分かれ目となります。エタニティは「重装」ユニットであり、チーム内でより多くの戦術的「重み」を占めることがよくあります。

標準的な編成では、前列に「プロテクター」（防御キャラ）を配置して攻撃を遮断し、エタニティは中列または後列に座ります。彼女の射程は通常中程度であるため、射程延長装備を使用しない限り、最後列に配置すると敵の後列を攻撃する能力が制限される可能性があります。

| 位置 | 役割 | 推奨バイオロイド | エタニティとのシナジー |
| :--- | :--- | :--- | :--- |
| **前列 (1列目)** | **メインタンク** | **サイクロプスプリンセス** | 攻撃を吸収し、カウンター攻撃を提供する |
| **中列 (2列目)** | **DPS (エタニティ)** | **エタニティ** | ビルドの核。直接攻撃から保護される |
| **後列 (3列目)** | **バッファー / ユーティリティ** | **マリア** | 重装ユニットにAPと攻撃バフを提供する |
| **どこでも** | **ターンコントロール** | **LRL** | 敵の速度を減らし、エタニティが先に動けるようにする |

<CardGrid cols={2}>
  <Card title="「重装」の核">

  </Card>
  <Card title="ターン順コントロール">
    **ソワン**や**LRL**のようなユニットは、敵のターンを遅らせることができます。これにより、エタニティは彼女の大規模なAOEキャノンをチャージするのに必要な時間を稼ぎます。
  </Card>
</CardGrid>
