<Section icon={Map} title="ラストオリジン R リロールティアリストのメカニクスを理解する">

**ラストオリジン R**の終末世界に足を踏み入れるプレイヤーにとって、「リロール」の概念は従来のガチャゲームとは大きく異なります。ほとんどのモバイルRPGでは、たった1つのメタを定義するユニットを引くために何時間もアカウントをリセットしますが、このゲームは低課金でカジュアルフレンドリーな哲学で設計されており、B、A、S、さらにはSSレアリティのバイオロイドやAGSユニットの大部分は、ストーリーステージやミッションを通じて無料で入手可能です。

**ラストオリジン R リロールティアリスト**は、「誰を引くか」よりも「誰を序盤に育成するか」に焦点を当てています。重機建設で高レアリティユニットを召喚しようとすることも可能ですが、初心者にとっては資源の無駄遣いと見なされることがよくあります。代わりに、メインストーリーをクリアして自動周回を解放し、オフライン中に資源やユニットを収集することに集中すべきです。

| 特徴 | 説明 | リロールへの影響 |
| :--- | :--- | :--- |
| ガチャスタイル | 低課金、スキン/スロットに焦点 | リロールは必須ではない |
| ユニット獲得 | ステージ/ミッションで周回可能 | 高ティアユニットは時間とともに保証される |
| 建設 | 資源を大量に消費する召喚 | 初心者にはリスキー；資源を温存する |
| 戦術訓練 | コアとなるメタユニットを報酬として提供 | 無料で最高のスタートチームを提供 |

<Callout type="tip" title="プロのヒント">
もし初期資源を使ってリロールするつもりなら、**No.2 ラバータ**や、序盤のクリアを早められる高レアリティのDPSユニットを探しましょう。しかし、戦術訓練ミッションに従うだけで、アカウントのリセットという手間をかけずに成功するために必要なすべてが得られます。
</Callout>

<Section icon={Swords} title="最強のスタートチーム：マングース部隊">

**マングースチーム**は、最も初心者向けのコアとして広く認識されています。この部隊のメンバーは全員、**戦術訓練室**ミッションを通じて入手できるため、トップティアのチームを構築するために運に頼る必要はありません。この部隊は、「キャリーを守る」メカニクスを利用しており、ミホが大量のダメージを与え、他のメンバーがサポート、アシスト攻撃、タンク役を提供します。

<Steps>
  <Step num="1" title="コアメンバーの獲得">
    戦術訓練ミッションを完了して、T-14 ミホ、T-60 ブルガサリ、AS-12 スチールドラコ、P-24 ピントをアンロックします。
  </Step>
  <Step num="2" title="戦闘優先度の設定">
    自動戦闘設定で、**ミホ**と**ブルガサリ**が**スキル2**を優先するように設定します。これにより、ダメージ出力とシナジーが最大化されます。
  </Step>
  <Step num="3" title="グリッド上の配置">
    スチールドラコを前列に配置して物理攻撃を受け止め、ピントは回避を活かせる位置に、ミホは保護された後列に配置します。
  </Step>
  <Step num="4" title="コピーの周回">
    ステージ**3-8**に進み、ミホの追加コピーを周回します。ユニットをリンクすることで、希少なコアを必要とせずにステータスが大幅に向上します。
  </Step>
</Steps>

| メンバー | ロール | 主要ステータス | スキル優先度 |
| :--- | :--- | :--- | :--- |
| **T-14 ミホ** | メインキャリー | ATK / クリティカル | スキル2（最優先） |
| **T-60 ブルガサリ** | サブDPS | ATK / クリティカル | スキル2（最優先） |
| **AS-12 スチールドラコ** | 防御タンク | 防御 / HP | スキル1（防御） |
| **P-24 ピント** | 回避タンク | 回避 | スキル1（アシスト） |

<Callout type="warning" title="重要事項">
**ピントの回避**を軽視しないでください。**ラストオリジン R**では、被弾率0%の回避タンクは、ダメージ軽減90%の防御タンクよりも優れていることがよくあります。高レベルステージでタンクが被弾すると、即座に倒される可能性が高いです。
</Callout>

<Section icon={Target} title="ステータス割り振り と 戦闘マスタリー">

新しいプレイヤーが犯す最大の過ちの1つは、ステータスポイントの誤った割り振りです。他のRPGではステータスのバランスを取るかもしれませんが、**ラストオリジン R**では特化が報われます。ダメージを与えるユニットであれば、そのダメージが確実に命中し、クリティカルヒットするように、すべてのポイントが割り振られる必要があります。

**ユニバーサルなステータスルール:**
- **DPSユニットの場合**: まず**クリティカル率100%**を達成する必要があります。この計算には、パッシブスキル、装備（チップ）、および基本ステータスが含まれます。100%に達したら、残りのポイントはすべて**ATK**に投入します。
- **回避タンクの場合**: 生存は二元的です。回避できなければ、死にます。何よりも**回避**を最大化してください。
- **防御タンクの場合**: **防御とHP**のバランスを取り、複数の物理攻撃に耐えられるようにします。

| ステータスタイプ | 目安 | 重要度 |
| :--- | :--- | :--- |
| **クリティカル率** | 100% | 安定したダメージを保証し、特定のパッシブを発動させる。 |
| **命中** | 300% | 後半のチャプターで高回避の敵に命中させるために必要。 |
| **装甲貫通** | 90%以上 | 敵のダメージ軽減を無視するために不可欠。 |
| **回避** | 最大 | 回避タンクがメタで機能するための唯一の方法。 |

<Callout type="info" title="戦闘モード">
ゲームには**オート/セミオート**モードと**マニュアル**モードがあります。初心者は資源を貯めるためにオート周回にこだわるべきです。マニュアルモードは、スキルタイミングとターゲット優先度が生存に不可欠な最も難しいエンドコンテンツでのみ必要です。
</Callout>

</Section>

<Section icon={Shield} title="資源管理とツナ缶のヒント">

**ツナ缶**は**ラストオリジン R**のプレミアム通貨です。バイオロイドの美しいスキンに使うのは魅力的かもしれませんが、序盤にそうすると進行が著しく妨げられます。プレイ開始後数ヶ月間は、アカウントのユーティリティが美学よりもはるかに価値があります。

**ツナ缶の優先的な使い道:**
1.  **拡張された部隊配置ベイ**: できるだけ早く5部隊に到達したいものです。これにより、周回中や睡眠中に、より多くのチームを資源探索に派遣できます。
2.  **キャラクター枠**: 周回するにつれて、インベントリがユニットでいっぱいになります。より多くの枠があれば、インベントリをクリアする頻度を減らせます。
3.  **装備枠**: キャラクター枠と同様に、EXステージで周回するSSティアの装備のためにスペースが必要です。
