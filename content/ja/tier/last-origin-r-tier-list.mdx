### コアゲームプレイサイクル
ゲームの主要なループは、ステージクリア、オートファーム機能の解放、コアチームの構築、リソースとユニットの収集、そしてより困難なチャレンジへの挑戦を含みます。多くのガチャゲームとは異なり、ラストオリジン R は支出が少なくカジュアルフレンドリーであり、コアとなるメタユニットはしばしばファーム可能です。

| 側面             | 説明                                                                 | ティアリストへの影響           |
| :----------------- | :-------------------------------------------------------------------------- | :---------------------------- |
| **ストーリー進行** | 新しいコンテンツ、ユニット、リソースを解放します。                                  | 序盤のユニットの入手しやすさ |
| **AFKリソースファーミング** | 一貫したリソース生成とキャラクター育成に不可欠です。     | ファームユニットの効率   |
| **キャラクター収集** | ほとんどのB/A/S/SSユニットはファーム可能です。メタユニットにガチャは必須ではありません。          | 無料ユニットの価値           |
| **チーム編成**  | グリッドベースの戦闘では、戦略的なユニット配置とシナジーが必要です。            | ユニットのシナジーと役割の有用性 |
| **オートバトル**    | オフラインファーミングやカジュアルプレイを可能にしますが、終盤には手動操作が必要です。   | ファームチームの使いやすさ    |

### ステータス配分ガイド
すべてのユニットが最適に機能するためには、特定のステータス閾値に達することが不可欠です。これらの閾値を無視することは、たとえ優れたユニットであっても性能が低下する一般的な理由であり、チーム全体の有効性に影響を与えます。

*   **ダメージディーラー (DPS)**: まず、**クリティカル率100%** (パッシブスキルと装備チップを含む) を達成します。この閾値に達したら、残りのすべてのポイントを**攻撃力**に割り当てます。
*   **回避タンク**: **回避**を最大化します。彼らの主要な生存メカニズムはダメージを避けることです。
*   **防御タンク**: **防御力**と**HP**の間でステータスポイントを分け、バランスの取れた耐久性のある前線を確保します。

<Callout type="info">
終盤のコアステータス閾値：クリティカル率 ≥100%、命中率 ≥300%、装甲貫通 ≥90%。進行するにつれてこれらを目指しましょう。
</Callout>

### 一般的な初心者の間違いとその回避方法
新規プレイヤーは、貴重なリソースを浪費し、進行を遅らせる罠にはまることがよくあります。これらを認識することで、時間と労力を大幅に節約できます。

| 間違い                        | 結果                                       | 解決策                                          |
| :----------------------------- | :------------------------------------------------ | :------------------------------------------------ |
| **無駄なツナ缶の消費** | 必要のないものにプレミアム通貨を浪費         | 展開、キャラクター、装備スロットを優先する |
| **SSユニットへの過度な依存**   | 構築が容易で効果的なB/Aティアチームの無視 | 序盤にはマングースチームを活用する          |
| **ステータス閾値の無視**   | ユニットの性能低下、全滅につながる              | クリティカル率100%、命中率300%、装甲貫通90%を達成する   |
| **ブラックブックの浪費**        | 非メタユニットや序盤ユニットの最大化           | 不敗のドラゴニュートのような終盤のメタユニットのために温存する |
| **終盤ステージの急ぎすぎ**     | リソースの損失、フラストレーション                        | まずオートファームとコアチームのセットアップをマスターする        |

<Callout type="warning" title="重要なアドバイス">
スキンバナーを引かないでください。これらは高価で非常にランダムであり、10連ガチャ1回でスキンを直接購入するのと同じくらいの費用がかかることがあります。まずゲームプレイの利点に焦点を当ててください。
</Callout>
