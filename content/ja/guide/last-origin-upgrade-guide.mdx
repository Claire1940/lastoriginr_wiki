### アップグレードのためのステータス配分ルール
ユニットの潜在能力を最大化するには、適切なステータス配分が不可欠です。すべてのユニットタイプについて、以下の一般的なルールに従ってください。

| ユニットタイプ | 主要ステータス | 副次ステータス | 備考 |
| :-------------- | :----------------- | :------------------- | :-------------------------------------------- |
| **ダメージディーラー (DPS)** | クリティカル率100% (パッシブ/チップを含む) | 残りのすべてのポイントを攻撃力に | 一貫したダメージ出力のために不可欠です。 |
| **回避タンク** | 回避最大化 | - | 彼らの生存は完全に高い回避に依存します。 |
| **防御タンク** | 防御力とHPをバランス良く | - | バランスの取れた前衛の耐久性を確保します。 |

<Callout type="warning" title="ステータス閾値の間違いを避ける">
クリティカル率100%、命中率300%、装甲貫通率90%などの重要なステータス閾値を無視すると、うまく育成されたチームでもより難しいコンテンツで失敗する可能性があります。常にユニットがこれらの最小値を満たしていることを確認してください。
</Callout>

### 初心者が避けるべきよくある間違い
多くの新規プレイヤーは、リソースを無駄にし、進行を遅らせる罠に陥りがちです。これらのよくある誤解に注意してください。

| 間違い | 進行への影響 | 正しいアプローチ |
| :----------------------------- | :------------------------------------ | :-------------------------------------------------------- |
| 無計画なツナ缶の消費 | プレミアム通貨の無駄遣い、成長の遅延 | 編成、キャラクター、装備スロットを優先する。 |
| SSユニットへの過度な依存 | 強力な無料のB/Aティアチームを無視する | マングースチームのようなコアチームを構築する。彼らが序盤を支えます。 |
| ステータス閾値を無視する | チームがより難しいコンテンツで失敗する | 必要な箇所でクリティカル率100%、命中率300%、装甲貫通率90%を確保する。 |
| ブラックブックの無駄遣い | 貴重なアイテムの最適でない使用 | 終盤のメタユニットのために温存する。序盤は控えめに使用する。 |
| 重工建設へのリソース投入 | 非効率的で危険なリソースの消耗 | オフライン回復と装備のために温存する。ゆっくりと建設する。 |
| ハードな終盤ステージへの突入 | 欲求不満、リソースの損失 | まずオートファームとコアチームのセットアップをマスターする。 |
