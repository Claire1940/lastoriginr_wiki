<Section icon={Map} title="9マス ラストオリジンR編成システムの理解">

ラストオリジンRの戦闘の核は、戦略的な3x3グリッドにあります。配置が純粋に見た目だけの従来のターン制RPGとは異なり、選択する**ラストオリジンR編成**は、どのユニットがバフを受け、どの敵をターゲットにできるか、そしてダメージが部隊全体にどのように分配されるかを直接決定します。すべてのユニットは、その座標または特定の味方との近接度に基づいて発動する独自のパッシブスキルセットを持っています。

個別のターンシステムでは、速度と配置が連携して機能します。速度の高いユニットは最初に行動しますが、その射程を妨げる編成に配置されている場合、その早期ターンは無駄になる可能性があります。グリッド座標（通常1から9まで番号が付けられています）を理解することが、ハイレベルなプレイへの第一歩です。

| グリッド位置 | 一般的な指定 | 主な役割の適合性 |
| :--- | :--- | :--- |
| **1, 4, 7** | 後列 | 長距離DPS、脆い支援、スナイパー |
| **2, 5, 8** | 中列 | バランス型アタッカー、オーラバッファー、オフタンク |
| **3, 6, 9** | 前列 | メインタンク、回避スペシャリスト、近接DPS |

<Callout type="warning" title="グリッドの脆弱性">
特定の敵スキルは「行」または「列」にダメージを与えます。特定のダメージ軽減がその配置で提供されない限り、すべての高優先度ユニットを単一の垂直または水平線に積み重ねることは避けてください。
</Callout>

</Section>

<Section icon={Shield} title="ユニットの役割と最適な配置戦略">

効果的な**ラストオリジンR編成**を構築するには、戦闘員を特定の役割に分類する必要があります。各役割には、生存率を最大化し、スキルが意図したターゲットに到達することを保証するグリッド上の「スイートスポット」があります。GOP Co., Ltd.版のゲームでは、すべての決定が重要であることが強調されており、これは戦闘が始まる前にユニットをどこにドラッグアンドドロップするかから始まります。

<CardGrid cols={3}>
  <Card title="防御型プロテクター">
    - **高HP/DEFステータス**
    - 通常は前列（3、6、9）に配置
    - 味方を直接的なターゲットから保護
  </Card>
  <Card title="戦術的サポーター">
    - **オーラベースのパッシブバフ**
    - 中央（5）または中列に配置
    - 周囲のAPと命中率を向上
  </Card>
  <Card title="攻撃型戦闘員">
    - **高ダメージ、低耐久性**
    - 後列（1、4、7）に配置
    - DPSを維持するために保護が必要
  </Card>
</CardGrid>

チームを配置する際には、「保護」メカニズムを考慮してください。多くのタンクユニットは、背後または隣接する味方の代わりにダメージを受けるパッシブ能力を持っています。メインのダメージディーラーが位置4にいる場合、タンクは安全網を提供するために理想的には位置5または7にいるべきです。

| 役割タイプ | ターゲット優先度 | 推奨グリッドスロット |
| :--- | :--- | :--- |
| **メインタンク** | 最高 | 6（中央前）または3/9 |
| **範囲DPS** | 中 | 4または7（後方隅） |
| **単体DPS** | 低 | 1（中央後方） |
| **バッファー** | 中 | 2、5、または8（中列） |

<Callout type="tip" title="回避 vs 防御">
回避タンクは単体ボスに対して前列でより良い性能を発揮し、防御重視のタンクは多くの小型敵がチッピングダメージを与えるステージで優れています。
</Callout>

</Section>

<Section icon={Target} title="ステップバイステップ：最初の競技用部隊の構築">

機能的な**ラストオリジンR編成**を作成するには、最強のユニットを選ぶだけでは不十分です。キャンペーンやイベントステージで見られる多様な課題に部隊が対処できるように、論理的な手順を踏む必要があります。

<Steps>
  <Step num="1" title="コアDPSを特定する">
    1つまたは2つの主要なダメージディーラーを選択します。彼らのスキル範囲を確認します。範囲が短い場合、中列に移動させる必要があるかもしれませんが、これによりリスクが増加しますが、敵の後列を攻撃できることが保証されます。
  </Step>
  <Step num="2" title="互換性のあるタンクを選択する">
    タンクをDPSに合わせます。DPSが回避ベースのユニットである場合、回避バフを提供するタンクは必須です。彼らが遅い場合、敵を挑発して時間を稼ぐことができるタンクを探します。
  </Step>
  <Step num="3" title="支援シナジーで埋める">
    ギャップを埋める支援を探します。部隊の命中率が不足している場合、「精密」オーラを提供するユニットを追加します。支援のバフ範囲がDPSとタンクが配置されているグリッドスロットをカバーしていることを確認します。
  </Step>
  <Step num="4" title="グリッドレイアウトを監査する">
    最終的な3x3レイアウトを確認します。脆いユニットが2つ以上無防備な状態になっていないことを確認します。ユニットがスロットに座っているが、チームの他の部分からパッシブな恩恵を全く受けていない「デッドゾーン」がないかを確認します。
  </Step>
</Steps>

<Callout type="success" title="編成検証">
高難易度コンテンツに挑む前に、必ず低コストのファーミングステージで新しい編成をテストしてください。ターン順序を観察し、アタッカーが行動する**前**に支援がバフをかけているかを確認してください。
</Callout>

</Section>

<Section icon={Swords} title="高度な編成アーキタイプとメタトレンド">

ラストオリジンRの2026年メタに進むにつれて、単純な配置だけではもはや不十分です。困難な敵の構成に対抗するために設計された特定のアーキタイプを採用する必要があります。これらのアーキタイプは、圧倒的な速度であろうと、破壊不可能な防御であろうと、特定の戦術的結果に力を集中させるためにグリッドを利用します。

| アーキタイプ名 | コア戦略 | 最適な用途 |
| :--- | :--- | :--- |
| **ザ・ダイヤモンド** | ユニットが5、2、8、4、6 | バランスの取れた保護とオーラの範囲 |
| **ザ・Tシールド** | ユニットが3、6、9、および5、2 | 1-2体の後列DPSのための最大前列ストール |
| **ザ・インバーテッドV** | ユニットが1、7、5、3、9 | AoEダメージの影響を最小限に抑えるために分散 |
| **ザ・スピードブリッツ** | すべてのユニットが中列/後列 | 敵が行動する前に一掃するための高AP集中 |

「ダイヤモンド」編成は、現在のところ一般的なキャンペーン攻略で最も人気があります。スロット5（中央）に高ティアの支援を配置することで、そのオーラは、ユニットがスロット2、4、6、8に配置されている場合、グリッド上の他のすべてのユニットに到達する可能性があります。これにより、リソース効率が最大化され、部隊全体が最高の性能で動作することが保証されます。

<Callout type="info" title="環境ハザード">
2026年の一部のイベントステージには、毎ターン特定のグリッド座標にダメージやデバフを与える「タイルハザード」があります。常にステージ情報を確認し、これらの危険なスロットを避けるように編成を調整してください。
</Callout>

</Section>
