### 必須の装備スロット

ラストオリジンR+のすべてのユニットには、特定の種類の装備のために設計された複数の装備スロットがあります。これらのスロットとそれに適合する装備を理解することが、ユニットを最適化するための第一歩です。

| 装備スロット | 主な機能                           | 強化されるステータスの例 |
| :-------------- | :--------------------------------------- | :-------------------------- |
| **武器**      | 攻撃力、クリティカルヒット率を増加      | ATK、CRT、ACC               |
| **防具**      | 防御力、HP、ダメージ軽減を向上     | DEF、HP、DMG耐性         |
| **チップ**      | ユーティリティ、速度、特定のステータスを強化  | SPD、ACC、回避、火力|
| **アクセサリー**| ユニークな効果、状態異常耐性を提供| 状態異常耐性、スキルダメージ |
| **補助装備**   | 追加の戦闘支援を提供         | AP獲得、リソース回復  |

</Section>

### 装備の入手場所

良い装備を入手する最も信頼できる方法は、ゲームプレイを通じてです。異なるステージやイベントでは、さまざまな種類と品質の装備が提供されます。

| 入手方法         | 入手できる装備の種類                  | 推奨戦略                  |
| :----------------- | :---------------------------------- | :------------------------------ |
| **ストーリーステージ**   | 基本～レア装備、製作材料| チャプターを進め、特定のドロップゾーンを周回 |
| **イベントステージ**   | イベント限定装備、高ティア材料| 期間限定イベントに参加し、すべての難易度をクリア |
| **補給作戦**   | 高ティア装備、強化モジュール   | 毎日プレイし、必要な特定の装備タイプに集中 |
| **製作**       | 集めた材料で特定の装備タイプを作成| 設計図を活用し、レア材料を優先 |
| **交換所**  | 特別イベント装備、レジェンダリーアイテム   | イベント通貨を貯め、ユニークなアイテムを優先 |

<Callout type="warning" title="リソース管理">
装備の強化には、ゴールドや強化モジュールなどのゲーム内リソースが大量に消費されます。リソースの枯渇を防ぐため、まずコアチームの装備強化を優先しましょう。
</Callout>

### セットボーナスの理解

多くの高ティア**ラストオリジン装備**はセットに属しています。同じセットの複数のピースを装備すると、強力な追加ボーナスが得られます。

| セットボーナスの種類 | 効果                                     | 理想的なユニット                      |
| :---------------- | :----------------------------------------- | :-------------------------------------- |
| **攻撃セット**    | 全体的な攻撃力をパーセンテージで増加   | ダメージディーラー、アタッカー              |
| **防御セット**    | 防御力とHPを向上                      | ディフェンダー、タンク、サポートユニット         |
| **速度セット**    | アクションポイント（AP）とターン順を増加| ユーティリティサポート、デバッファー             |
| **クリティカルセット**  | クリティカルヒット率とダメージを強化      | 高ATKダメージディーラー、スナイパー        |
| **回復セット**    | 受ける回復量または与える回復量を向上         | ヒーラー、自己回復ユニット          |

<Callout type="info" title="シナジーが鍵">
装備を選ぶ際は、常にユニットの基本ステータスとスキルを考慮してください。ディフェンダーは防御セットからより恩恵を受けるかもしれませんが、アタッカーは攻撃セットまたはクリティカルセットで活躍するでしょう。
</Callout>

### サブステータスの優先順位付け

主要なステータス以外に、装備にはランダムなサブステータスが付与されます。これらはユニットの性能に大きな影響を与える可能性があります。

| ユニットの役割    | 優先すべきサブステータス             | 避けるべきステータス                |
| :-------------- | :----------------------------------------- | :---------------------------------- |
| **ダメージディーラー** | クリティカル率、クリティカルダメージ、攻撃力%、命中 | 回避、防御力%、HP%              |
| **タンク/ディフェンダー** | HP%、防御力%、ダメージ耐性、回避      | クリティカル率、クリティカルダメージ、攻撃力% |
| **サポート**     | 速度、命中、効果抵抗、AP獲得| クリティカルダメージ、攻撃力%            |
| **ヒーラー**      | 回復量、HP%、効果抵抗     | 攻撃力%、クリティカル率              |

</Section>
